## Description

This is a simple ecommerce API that uses the NestJS framework.
- It allows users onboard, get athenticated and create products.
- It also allows unauthenticated users to view list of approved products.
- A dedicated endpoint has been created to allow admin user account creation.

## Project setup

```bash
$ git clone https://github.com/timiFoxtrot/farooq-nest-ecommerce.git
$ cd farooq-nest-ecommerce
```

## Install dependencies

```bash
$ npm install
```

## Add the environment variables

```bash
PORT=4420
NODE_ENV=development
ENABLE_DOCUMENTATION=true
APP_URL=http://localhost

DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=maxuser
DB_PASSWORD=password
DB_DATABASE=postgres
DB_SCHEMA=ecommerce

JWT_SECRET=yoursecretisafewithme
JWT_EXPIRY='24h'
AUTH_ENGINE=JWT

ADMIN_NAME='Super Admin'
ADMIN_PASSWORD=AdminPass11..
ADMIN_EMAIL=super.admin@yopmail.com
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Swagger Documentation

```bash
$ http://localhost:4420/api
```
